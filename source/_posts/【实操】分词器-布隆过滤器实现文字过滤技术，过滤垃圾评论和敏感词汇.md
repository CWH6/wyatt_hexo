# HanLP+布隆过滤器过滤敏感词

## HanLP

称为`汉语言处理包`，是一套`自然语言处理工具包`，其中包括了许多常用的中文处理工具，例如`分词`、词性标注、命名实体识别等。它是由中国科学院计算技术研究所自然语言处理与社会人文计算研究中心开发的开源项目。

**常用的功能**

1. **分词器**：HanLP 提供了多种分词器，包括了基于统计的分词器、基于感知机的分词器等，可以满足不同场景的需求。
2. **词性标注**：能够将句子中的每个词语标注上相应的词性，如名词、动词、形容词等。
3. **命名实体识别**：能够识别出句子中的命名实体，如人名、地名、组织名等。
4. **依存句法分析**：可以分析句子中词语之间的依存关系，找出句子的主谓宾结构。
5. **关键词提取**：根据句子的内容，提取出关键词。
6. **文本分类**：可以将文本按照预先定义的类别进行分类。
7. **短文本相似度计算**：可以计算两段文本之间的相似度。
8. **繁简转换**：可以将繁体字转换为简体字，反之亦然。
9. **拼音转换**：可以将汉字转换为拼音。
10. **自定义词典**：可以自定义词典，以满足特定领域的分词需求。

## 敏感词屏蔽

```java
public class SensitiveWordsFilter {


    public static void main(String[] args) {

        // 创建一个布隆过滤器，预期插入数为 1000，误判率为 0.01
        BloomFilter<CharSequence> bloomFilter = getBloomFilter(1000, 0.01);

        // 分词器配置
        Segment shortestSegment = getSegment();

        // 对句子分词，分词结果展示如下:
        String text = "我是程序员,我喜欢原神,云顶之弈,许嵩";
        List<Term> termList = shortestSegment.seg(text);
        termList.stream().forEach(term -> {
            System.out.println(term.word);
        });

        // 根据布隆过滤器与词语匹配，如果为敏感词则设置为等长度的“*”
        StringBuilder resultBuilder = new StringBuilder();
        termList.stream().forEach(term -> {
            String word = term.word;
            if (bloomFilter.mightContain(word)) {
                resultBuilder.append(fixWork(word));
            } else {
                resultBuilder.append(word);
            }
        });
        System.out.println("原文字：" + text);
        System.out.println("修改后：" + resultBuilder);
    }


    /**
     * 创建分词器
     *
     * @return
     */
    private static Segment getSegment() {
        return new DijkstraSegment()
                // 禁用自定义词典，这表示不会使用用户自定义的词库进行分词。
                .enableCustomDictionary(false)
                // 启用地名识别，可以识别出句子中的地名
                .enablePlaceRecognize(true)
                // 启用机构名识别，可以识别出句子中的机构名
                .enableOrganizationRecognize(true); 
    }

    /**
     * 创建一个布隆过滤器
     *
     * @param wordSize 敏感词个数
     * @param rate     误判率
     * @return
     */
    private static BloomFilter<CharSequence> getBloomFilter(int wordSize, double rate) {
        BloomFilter<CharSequence> bloomFilter = BloomFilter.create(
                Funnels.stringFunnel(Charset.forName("UTF-8")), wordSize, rate);

        // 布隆过滤器添设置敏感词
        bloomFilter.put("hello");
        bloomFilter.put("程序员");
        bloomFilter.put("许嵩");

        return bloomFilter;
    }

    /**
     * 将敏感词则设置为等长度的“*”
     *
     * @param work 敏感词
     * @return
     */
    private static String fixWork(String work) {
        String res = "";
        for (int i = 0; i < work.length(); i++) {
            res += "*";
        }
        return res;
    }
}
```
